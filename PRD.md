# AI 주식 분석 어시스턴트 (AI Stock Intelligence Platform) — PRD

**작성일**: 2026-06-24  
**상태**: MVP 기획 최종 완료 (개선 + 평가)  
**팀 규모**: 1명 (단독)

---

## 📋 프로젝트 개요

초보 개인투자자를 위한 AI 기반 투자 의사결정 지원 플랫폼

사용자가 등록한 관심종목(최대 50개)을 대상으로 AI가 기술적 분석, 수급 분석, 뉴스, 펀더멘털, 거시경제를 종합 분석하여 **매수가/매도가 추천**을 제공한다.

추천 성과를 자동으로 추적하고 AI 스스로 분석 가중치를 개선하는 **자기개선 피드백 루프**를 통해 시간이 지날수록 정확도를 높인다.

## 🎯 핵심 원칙

### 1. Explainable AI
모든 추천은 반드시 근거를 제공한다
```
매수 추천
근거
- 상승 추세 유지
- 외국인 순매수 증가
- 거래량 급증
- 반도체 업종 강세
- 긍정 뉴스 증가
```

### 2. Confidence Score
모든 분석 결과에 신뢰도를 표시한다
```
매수 의견
투자 점수: 84/100
신뢰도: 91%
```

### 3. 시나리오 기반 분석
단일 목표가 대신 확률 기반 시나리오 제시
```
상승 시나리오: 58% | 목표가 95,000원
횡보 시나리오: 27% | 가격범위 82,000~88,000원
하락 시나리오: 15% | 손절가 78,000원
```

### 4. 이중 관점 분석
**단기 관점(1~3개월)**: 기술적 분석, 차트 패턴 중심  
**장기 관점(6개월~1년)**: 펀더멘털, 산업 전망 중심

### 5. 단기/장기 상충 신호 처리
단기와 장기 추천이 서로 반대일 때(예: 단기 매도 / 장기 매수),
AI가 가중치를 고려하여 **하나의 종합 의견**으로 정리한다.

### 6. 낮은 신뢰도 처리 정책
신뢰도 60% 이하일 경우 **숨기지 않고 그대로 표시**하되
⚠️ "신뢰도 낮음 — 참고용으로만 활용 권장" 경고를 함께 표시한다.

## 🏗️ 분석 엔진 구조 (11 Layer)

### Layer 1: Market Regime
현재 시장 상태 분석 (강세장/약세장/횡보장)
- 코스피, 코스닥, S&P500, VIX, 국채금리, 환율

### Layer 2: Smart Money Analysis
기관 및 외국인 자금 흐름 분석
- 외국인 순매수, 기관 순매수, 공매도, 대차잔고

### Layer 3: Market Structure
상승/하락/붕괴 추세 판단
- Higher High/Low vs Lower High/Low

### Layer 4: Volume Profile
거래량 기반 지지·저항 분석
- 강한 지지구간, 강한 저항구간

### Layer 5: Technical Indicators
MA20/60/120/240, RSI, MACD, Bollinger Bands, ATR, ADX, OBV, VWAP
- 단순 계산이 아니라 종합 해석

### Layer 6: Candlestick Analysis
최근 30~60개 캔들 패턴 분석
- Hammer, Morning Star, Bullish Engulfing, Shooting Star 등

### Layer 7: Chart Pattern Recognition
자동 패턴 탐지
- Cup and Handle, Head and Shoulders, Triangle, Wedge, Box Range 등

### Layer 8: Sector Rotation
업종 강도 분석
- 반도체, AI, 바이오, 방산, 전기차, 2차전지 등

### Layer 9: Fundamental Analysis
기업 가치 분석
- PER, PBR, ROE, EPS, 영업이익, 부채비율, 매출 성장률, FCF

### Layer 10: News Analysis
뉴스 및 공시 감성 분석
- 긍정/중립/부정 판정 및 감성 점수

### Layer 11: Macro & Global Analysis
국제 정세 분석
- 미국 금리, 연준 정책, CPI, 달러 인덱스, 원유, 지정학 리스크

**사용자 화면에서 보는 것**
- 투자 가능 금액 입력 (예: 1000만원)
- 종목별 매수가/매도가 추천
- 단기 관점, 장기 관점
- **AI 포트폴리오 누적 수익률** (%)

**백그라운드에서 자동 실행**
- AI가 추천대로 자동 매수/매도 **시뮬레이션** 실행

- 추천 성과 추적:
  - 매수가에 도달했는가?
  - 매도가를 달성했는가?
  - 손절가에 걸렸는가?
- 성과 기반 11개 레이어 가중치 자동 조정
- 조정 주기: **2일마다** (지속적 개선)

## 📦 데이터 저장 원칙

### OHLCV 데이터
- 관심종목 전체에 대해 **최소 10년치 OHLCV 데이터** 저장
- 저장 단위: 일봉(1D), 주봉(1W), 월봉(1M)
- DB 저장 후 주기적(일 1회) 갱신

### 타임프레임 자동 선택
AI 분석 엔진은 분석 목적에 따라 타임프레임을 자동 선택한다:

| 분석 목적 | 사용 타임프레임 |
|-----------|----------------|
| 단기 관점 (1~3개월) | 일봉(1D) 위주 |
| 중기 관점 (3~6개월) | 일봉(1D) + 주봉(1W) |
| 장기 관점 (6개월~1년+) | 주봉(1W) + 월봉(1M) |
| 시장 구조 분석 (Layer 3) | 주봉(1W) + 월봉(1M) |
| 기술적 지표 (Layer 5) | 일봉(1D) |
| 차트 패턴 인식 (Layer 7) | 일봉(1D) + 주봉(1W) |

## 🔄 AI 자기개선 피드백 루프 (Self-Improving System)

### 핵심 원칙
- **실제 매매는 하지 않음** (모의 투자만)
- **2일마다 가중치 자동 조정** (지속적 개선)
- **추천 성과를 자동으로 추적** (매수가 도달? 매도가 달성? 손절?)

### 피드백 루프 흐름

```
1️⃣ AI 추천 생성
   └─ "삼성전자, 매수가 80,000원 / 매도가 88,000원"
   └─ AI 점수: 88점, 신뢰도: 92%

2️⃣ 추천 기록 저장 (DB)
   └─ 종목, 추천일, 매수가, 매도가, AI점수, 신뢰도
   └─ 각 레이어별 점수 스냅샷 저장

3️⃣ AI 포트폴리오에서 자동 모의 투자
   └─ 추천대로 시뮬레이션 매수/매도 실행
   └─ 실제 시장 데이터로 수익률 계산

4️⃣ 실제 시장 결과 자동 추적
   ├─ 30일 후: 최저가가 매수가에 도달했는가?
   └─ 90일 후: 최고가가 매도가에 도달했는가? 또는 손절?

5️⃣ 결과 분류
   ├─ 성공 ✅: 매수가 도달 + 매도가 달성 (수익 실현)
   ├─ 손절 ⚠️: 매수가 도달 + 손절가 먼저 터치 (손실)
   └─ 미도달 ⏳: 매수가 자체 미도달 (진입 실패)

6️⃣ 레이어별 기여도 분석
   └─ 성공한 추천: 어떤 레이어가 정확했는가?
   └─ 실패한 추천: 어떤 레이어가 틀렸는가?

7️⃣ 가중치 자동 조정 (2일 주기, 지속)
   매 2일마다:
   ├─ 성공 예측 레이어 → 가중치 증가 (+2% 이내)
   ├─ 실패 예측 레이어 → 가중치 감소 (-2% 이내)
   └─ 급격한 변동 방지: ±2% 제한, 계속 반복

8️⃣ 다음 추천에 반영
   └─ 개선된 가중치로 더 정확한 매수가/매도가 추천
```

### 자기개선 로직 예시

**2일마다 반복 조정**

```
[2일차]
누적 추천 데이터 (처음 2일):

- Layer 2 (Smart Money): 5개 추천 중 3개 성공 → 60% 정확도

전: Layer 1 = 16%, Layer 2 = 14%

```
